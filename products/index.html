Sudopoint (sudopoint.com)

Sudopoint is a web platform for generating test datasets, primarily for software testers and demo’ers. It provide a no-code platform where users can buy datasets or pay a monthly subscription.

I’m writing a book on how to get started with PostgreSQL with Newline, a modern “O’Reilly like” publishing platform. As part of my book preparation, I taught a SQL class to the public in early 2021 and a Python class as part of Code in Place. I’m looking for more opportunities to teach SQL classes to folks in industry - if you’re interested, please get in touch!
